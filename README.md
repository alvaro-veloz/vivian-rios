# Dra. Vivian Ríos Gómez — Médica

Premium landing page for a medical doctor featuring an interactive Spline 3D hero, scroll-triggered animations, and a WhatsApp-native appointment booking flow. Designed to communicate clinical trust and modern professionalism in a single, high-impact page.

## Features

- Interactive Spline 3D model embedded as full-screen hero
- Scroll-triggered animations throughout
- WhatsApp CTA for direct appointment booking
- Open Graph image included for rich social previews
- CSS backup layer for Spline-free fallback (styles.css.no movimiento)
- Fully responsive, mobile-first layout
- No frameworks — pure HTML, CSS, and JavaScript

## Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (custom, no frameworks) |
| Logic | Vanilla JavaScript (ES6+) |
| 3D Hero | Spline (spline.design) |
| Images | WebP optimized |
| Hosting | Cloudflare Pages |
